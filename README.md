# CV & Tabular Machine Learning Portfolio

This repository contains Machine Learning projects:
1. **Titanic Survival Prediction** (Random Forest, Feature Engineering)
2. **Dogs vs. Cats Image Classification** (PyTorch CNN, Transfer Learning with ResNet18)

## Project 1: Titanic Survival
* **Goal:** Predict passenger survival based on tabular data.
* **Tech Stack:** Pandas, Scikit-Learn, Seaborn.
* **Dataset:** Download the `train.csv` and `test.csv` from [Kaggle Titanic](https://www.kaggle.com/c/titanic/data).

## Project 2: Dogs vs. Cats
* **Goal:** Binary image classification of messy, real-world images.
* **Tech Stack:** PyTorch, Torchvision.
* **Dataset:** Download the dataset from [Kaggle Dogs vs Cats](https://www.kaggle.com/c/dogs-vs-cats/data) and extract it into a `data/train/` and `test/` folder structure.

## How to Run
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Download the datasets as instructed above.
4. Run the Jupyter Notebooks.
